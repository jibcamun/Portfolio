# Jibran Noorshah's Portfolio

Personal portfolio and blog built with Next.js, MDX, and Tailwind CSS.

## Features

- Portfolio homepage
- MDX blog posts
- RSS feed
- Sitemap and robots metadata
- Dynamic Open Graph images
- Syntax highlighting
- Vercel Analytics and Speed Insights

## Development

```bash
pnpm install
pnpm dev
```
